# Support

> Where to go for help, by what you're trying to do.

## You're using AZMX and something's wrong

1. Skim [`TROUBLESHOOTING.md`](TROUBLESHOOTING.md) — common issues + fixes.
2. Check [`FAQ.md`](../FAQ.md) — broader Q&A.
3. Search [existing issues](https://github.com/AzmxAI/azmx/issues?q=is%3Aissue) for someone who hit the same thing.
4. Still stuck? [Open a bug report](https://github.com/AzmxAI/azmx/issues/new/choose).

## You want to talk about a feature

- [Open a Feature request](https://github.com/AzmxAI/azmx/issues/new/choose) — concrete user pain, then proposed shape.
- For broader ideas, [start a Discussion](https://github.com/AzmxAI/azmx/discussions).

## You found a security issue

**security@azmx.app** — privately. Never as a public issue. Acknowledged within 24h, status update within 7 days. See [`SECURITY.md`](../SECURITY.md).

## You want to buy Pro / Teams / Enterprise

- **Pro** — self-serve at [azmx.ai/pricing](https://azmx.ai/pricing). Monthly or annual.
- **Teams** — also self-serve at [azmx.ai/pricing](https://azmx.ai/pricing).
- **Enterprise / Gov / Air-gapped / SOC 2 + DPA + SBOM** — **sales@azmx.app**.

## You have a billing question

**billing@azmx.app** — questions about charges, refunds, plan changes, receipts. Payment is processed by Polar.sh (our Merchant of Record); they can also help via [polar.sh/help](https://polar.sh/help).

## You're a journalist / analyst / podcaster

**press@azmx.app** — interviews, demo accounts, briefings, brand assets. We're happy to help; quick replies guaranteed during a publication cycle.

## You're partnering / integrating

**partners@azmx.app** — connector integrations, OEM, white-label, embed.

## You're a researcher

**security@azmx.app** for any security research; **research@azmx.app** for academic interest (we sometimes give free Enterprise licenses for academic security research).

## You're contributing to the docs

PRs welcome on this repo. See [`CONTRIBUTING.md`](../CONTRIBUTING.md).

## You want updates

- **Releases** → [GitHub releases](https://github.com/AzmxAI/azmx/releases) (every tagged version, with notes).
- **Blog** → [azmx.ai/blog](https://azmx.ai/blog) (engineering write-ups, design notes).
- **The roadmap** → [`ROADMAP.md`](ROADMAP.md).

## What we don't offer

- 24/7 paid phone support (yet — coming with Enterprise SLA).
- Custom development for hire.
- A free Slack / Discord — we may run one later; until then, GitHub Discussions is the place.

## Response times

| Channel | Target |
|---|---|
| security@azmx.app | 24h ack, 7d update |
| sales@azmx.app | 1 business day |
| GitHub issues | Triage within 2 business days |
| GitHub Discussions | Best-effort, community + maintainers |
| Twitter / social | Not a support channel |

We're a small team. Patience appreciated.
